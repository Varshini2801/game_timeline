Time Traveler's Security Challenge

Welcome, Time Agent! A temporal anomaly has fractured our timeline, and phishing scams from all eras are leaking through!

Your mission: identify phishing attempts across three time periods to collect the fragments of a temporal access code. Only by completing all three levels will you restore the timeline.

How to play:

  • Each level features phishing examples from a different time period
  • Examine each example carefully and determine if it's "Phish" or "Legit"
  • Each correct answer earns you a letter of the temporal access code
  • After completing all questions in a level, use the collected letters to form the password
  • Enter the correct password to proceed to the next era

Remember: Look for suspicious URLs, poor grammar, urgent requests, generic greetings, and unsecured connections!
